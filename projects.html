Descubre Nuestros Proyectos Bienvenido a nuestra sección de proyectos. Aquí, te invitamos a explorar tres ejemplos representativos de nuestro trabajo: Una página personal diseñada con HTML y CSS. Una aplicación de lista de tareas creada con JavaScript. Una galería de imágenes interactiva desarrollada con jQuery. ¡Descubre cómo combinamos tecnología y creatividad para crear soluciones únicas y atractivas!

  • HTML, CSS : Una página web que muestra tu portafolio personal. Incluye secciones como "Sobre mí", "Proyectos", "Habilidades" y "Contacto". Utiliza HTML para estructurar el contenido y CSS para estilizarlo y crear un diseño atractivo y responsivo.

  • JavaScript: Una aplicación web simple que permite a los usuarios crear, editar y eliminar tareas. Utiliza JavaScript para manejar la lógica de la aplicación, como agregar y eliminar elementos de la lista de tareas, así como para interactuar con el DOM para actualizar la interfaz de usuario en tiempo real.

  • jQuery: Una galería de imágenes interactiva que permite a los usuarios navegar por una colección de imágenes utilizando controles de navegación, como botones de siguiente y anterior, así como miniaturas de las imágenes. Utiliza jQuery para simplificar la manipulación del DOM y añadir efectos de animación para mejorar la experiencia del usuario.
