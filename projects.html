Proyectos

Project 1: Página Personal

Una página personal diseñada con HTML y CSS. Incluye secciones como "Sobre mí", "Proyectos", "Habilidades" y "Contacto". Utiliza HTML para estructurar el contenido y CSS para estilizarlo y crear un diseño atractivo y responsivo.

Project 2: Aplicación de Tareas

Una aplicación web simple que permite a los usuarios crear, editar y eliminar tareas. Utiliza JavaScript para manejar la lógica de la aplicación, como agregar y eliminar elementos de la lista de tareas, así como para interactuar con el DOM para actualizar la interfaz de usuario en tiempo real.

Project 3: Galería de Imágenes

Una galería de imágenes interactiva que permite a los usuarios navegar por una colección de imágenes utilizando controles de navegación, como botones de siguiente y anterior, así como miniaturas de las imágenes. Utiliza jQuery para simplificar la manipulación del DOM y añadir efectos de animación para mejorar la experiencia del usuario.
